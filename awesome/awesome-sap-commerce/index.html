    Awesome SAP Commerce ¶

    A curated list of delightful SAP Commerce (formerly Hybris) resources. SAP Commerce is an e-commerce platform using Java, based on Spring MVC framework. The useful resources for SAP Commerce Developers are below.

    SAP Commerce Resources¶

    Official Sites¶

    • Help SAP Commerce - Provides official SAP Commerce documentation .
    • Community SAP Commerce Cloud - Featured content about SAP Commerce.
    • SAP Commerce Terminology - Familiarize yourself with the terminology used in SAP Commerce Cloud documentation.
    • Learning SAP Commerce Cloud - Provides technical essentials of SAP Commerce.
    • Learning Journeys - Provides Learning journey for SAP Commerce developer role.
    • SAP Learning Hub - Subscription required techinal videos, articles.
    • SAP Store - Several related extensions and add-ons for SAP Commerce. Some of them provides a free trial for a deep dive.
    • Support SAP Commerce - Get support from the expert service team.
    • SAP Commerce Newsletter - Newsletter that monthly news delivered directly to your inbox.

    This website contains SAP Commerce zip files, with new versions released every year. Before downloading, important to know which version your customer is using.

    • Download SAP Commerce - S-user is required to login.

    These two websites are going to migrate to Learning SAP on July 2024.

    • SAP Micro Learning - Login required.
    • Open SAP Courses - Some free courses about SAP Commerce.

    Related Technologies¶

    Ant¶

    Apache Ant is the default code build and source management tool.

    • Apache Ant - Official Ant web site.

    Groovy¶

    Apache Groovy is a functional programming language, that can be possible to run scripts.

    • Tutorials Point - Groovy tutorials to understand basic syntax.

    Solr¶

    Apache Solr is the default search engine.

    • Apache Solr - Official Solr web site.
    • Yonik Solr - Solr syntax explained by Yonik Seeley.

    Spartacus¶

    Spartacus is a lean Angular-based JavaScript storefront for SAP Commerce Cloud that replaced JSP Storefront. Recently, It re-branded as SAP Composable Storefront.

    • Spartacus Documentation
    • Vercel Integration for SAP Composable Storefront
    • Angular 101 Medium
