    Awesome Steam Deck ¶

    A curated list of awesome Steam Deck software, resources and more.

    Steam Deck is a handheld gaming computer made by Valve.

    Contributions and suggestions about how to improve this project are welcome!
    Please follow our contribution guidelines.

    If you want to support my work you can do it following me, leaving a star, sharing my projects or also donating at the links below.
    Choose what you find more suitable for you:

    Software¶

    • Distrobox - Use any Linux distribution inside your terminal.
    • rwfus - Read and write OverlayFS for Steam Deck.
    • BoilR - Synchronize non-Steam games with your Steam library.
    • Steam ROM Manager - An app for managing ROMs in Steam.
    • Discord Overlay - Overlay for Discord on Steam.
    • SteamOS Btrfs - Convert the /home partition from ext4 to btrfs.
    • ProtonUp-Qt - Install and manage third-party compatibility tools for Steam Games.
    • Clover - Graphical boot manager for the Steam Deck.
    • Steam Deck Printing Rootless - Utility that lets you print on the Steam Deck rootlessly and that survives updates as well.
    • Protontricks - Configure Proton prefixes for Steam games.
    • SteamTinkerLaunch - Advanced game launch customization and tweaking.
    • GOverlay - Graphical UI for MangoHud configuration.
    • CoreCtrl - Advanced CPU/GPU control utility.
    • Flatseal - Manage Flatpak permissions with a GUI.
    • SteamGridDB Manager - Apply artwork for non-Steam games automatically.
    • Warehouse - Modern Flatpak management tool.

    Performance¶

    • MangoHud - Vulkan/OpenGL performance overlay (FPS, temps, CPU/GPU usage).
    • Gamescope - Micro-compositor for resolution scaling, frame limiting, and latency improvements.
    • PowerTools - Decky plugin for advanced CPU, GPU, and power management.
    • SimpleDeckyTDP - Decky plugin with simplified TDP controls for Steam Deck.
    • ShareDeck - Community database of Steam Deck performance settings for games.
    • HandheldHQ - Per-game Steam Deck settings cross-checked from multiple guides, with tested FPS and battery.

    Customization¶

    • CSS Loader - Decky plugin to apply custom themes to Steam Deck UI.
    • Audio Loader - Decky plugin for custom boot and suspend sounds.
    • Steam Deck Repo - Custom boot videos and animations.

    Power Management¶

    • PowerControl - Decky plugin for advanced power profile management.

    • EmuDeck - Emulator configurator.
    • RetroArch - Frontend for emulators, game engines and media players.
    • RetroDECK - All-in-one sandboxed application to play your retro games.
    • EmulationStation DE - Frontend for browsing and launching games from various collections.
    • PCSX2 - PS2 Emulator.
    • RPCS3 - PS3 Emulator.
    • shadPS4 - PS4 Emulator.
    • PPSSPP - PSP Emulator.
    • Vita3K - PS Vita Emulator.
    • DuckStation - Fast and accurate PlayStation 1 emulator.
    • bsnes - SNES Emulator.
    • Dolphin - Gamecube/Wii Emulator.
    • DeSmuME - Nintendo DS Emulator.
    • melonDS - Nintendo DS emulator with local multiplayer support.
    • Cemu - Wii U emulator with excellent compatibility.
    • xemu - Xbox Emulator.
    • MAME - Arcade machine emulator.
    • Flycast - Sega Dreamcast/Naomi emulator.

    Launcher¶

    • Lutris - Open gaming platform.
    • Heroic - Epic Games and GOG launcher.
    • Bottles - Run Windows software and games under Linux.
    • NonSteamLaunchers - Automatic installation of the most popular launchers.
    • Minigalaxy - Lightweight GOG client for Linux.
    • Prism Launcher - Minecraft launcher with mod support.
    • Itch - Indie game platform client.
    • Legendary - CLI launcher for Epic Games Store.

    Remote¶

    • Chiaki - Free and open source software client for PlayStation 4 and PlayStation 5 Remote Play.
    • KDE Connect - Multi-platform app that allows your devices to communicate.
    • LocalSend - Open source app to share files between devices over your local network.
    • Steam Link - Extend your Steam gaming experience to more devices.
    • Greenlight - An open source client for streaming Xbox Cloud and Xbox Home.
    • Moonlight - Stream games from PC with NVIDIA GPU.
    • Sunshine - Self-hosted game stream host (pairs with Moonlight).
    • Parsec - Low-latency game streaming and co-op gaming.

    File Management¶

    • Deck Drive Manager - Copy PC games from PC to Steam Deck drives.
    • Syncthing - Open source continuous file synchronization.
    • OpenCloudSaves - A tool used for syncing your save games across your devices.

    Storage Management¶

    • Storage Cleaner - Decky plugin to clean shader cache and temporary files.
    • CompSize - Check btrfs compression ratios on Steam Deck.

    Utility¶

    • Shortix - A script that creates human readable symlinks for Proton game prefixes.
    • Cryoutilities - Scripts and utilities to improve performance and manage storage.
    • Steam Deck Shader Cache Killer - Script to purge Steam Deck shader cache.
    • Ludusavi - Backup tool for PC game saves.

    Accessibility¶

    • Orca Screen Reader - Screen reader for desktop mode.
    • Magnus - Screen magnifier utility for better visibility.
    • Text-to-Speech Tools - System TTS for accessibility needs.

    Plugin¶

    • Decky Loader - Plugin launcher.
    • Deckbrew Plugins - List of Deckbrew plugins.
    • PowerTools - Advanced CPU, GPU, and power management.
    • VibrantDeck - Control display saturation and vibrance.
    • Pause Games - Pause any game, even those without native pause functionality.
    • AutoFlatpaks - Automatically update Flatpak applications.
    • SteamGridDB Plugin - Browse and apply custom game artwork directly from Steam Deck.
    • AutoSuspend - Automatically suspend Steam Deck after inactivity period.
    • HLTB for Deck - How Long to Beat integration showing game completion times.
    • MagicPods - A magic plugin for Decky Loader that lets you control your AirPods and Beats headphones in a comfortable way.
